# Employee Data Analysis Using Pandas

This project analyzes an employee dataset containing around 10,000 records.

## Dataset Features
- Employee_ID
- Name
- Department
- Age
- Gender
- Salary
- Experience_Years
- City
- Joining_Date

## Analysis Performed
- Data cleaning and missing value handling
- Salary statistics (min, max, average)
- Department-wise employee count
- Gender distribution
- City-wise employee distribution
- Experience analysis

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
